PIMD Series - Part II

NVT Thermostats and GLE

2026.05.27 - NVT / PILE / GLE / PIGLET

I. Ring-Polymer Basics II. NVT and GLE III. Cayley Stability

Part I built the ring-polymer Hamiltonian. The next practical question is how to sample it efficiently. This note separates three related ideas: ordinary NVT Langevin sampling, PILE-style normal-mode thermostats, and GLE/PIGLET-style covariance control for reducing the bead count needed for selected quantum observables.

Why NVT is not optional

The ring polymer contains a broad band of internal frequencies,

\[ \omega_k=2\omega_P\sin\left(\frac{k\pi}{P}\right), \qquad \omega_P=\frac{P}{\beta\hbar}. \]

As \(P\) grows, the highest internal modes become stiff. Microcanonical propagation can conserve the extended Hamiltonian while still sampling slowly or non-ergodically. A thermostat is therefore not just a temperature-control accessory; it is part of the sampling algorithm.

PILE in normal modes

For a harmonic physical potential, each ring-polymer normal mode has an effective frequency

\[ \Omega_k=\sqrt{\omega_k^2+\omega_0^2}. \]

A simple white-noise Langevin thermostat uses one friction for all modes. PILE instead thermostats the normal modes locally, often choosing a friction of order the mode frequency:

\[ \dot Q_k=\frac{P_k}{m}, \qquad \dot P_k=-m\Omega_k^2Q_k-\gamma_k P_k +\sqrt{\frac{2m\gamma_k}{\beta_P}}\,\eta_k(t), \qquad \gamma_k\approx 2\Omega_k. \]

The centroid mode is usually handled more gently because it is the physically meaningful low-frequency component; the internal modes are sampling variables and can be thermostatted aggressively.

The useful diagnostic is the dimensionless ratio \(\gamma_k/(2\Omega_k)\). A single friction can be reasonable for one frequency band but mismatched for the rest of the ring-polymer spectrum. A PILE-like choice keeps this ratio near unity for the internal modes, so each artificial mode is damped on its own natural timescale. This changes how quickly the canonical finite-\(P\) distribution is sampled; it does not remove the finite-\(P\) discretization bias.

GLE and PIGLET

A generalized Langevin equation introduces auxiliary momenta so that a non-Markovian memory kernel becomes a Markovian system in an enlarged space:

\[ \dot q=\frac{p}{m}, \qquad \begin{bmatrix}\dot p\\ \dot{\mathbf s}\end{bmatrix} = \begin{bmatrix}-\nabla V(q)\\ \mathbf 0\end{bmatrix} - \mathbf A_p \begin{bmatrix}p\\ \mathbf s\end{bmatrix} + \mathbf B_p\boldsymbol\xi(t). \]

For canonical sampling, the fluctuation-dissipation constraint fixes the noise covariance, commonly written as

\[ \mathbf B_p\mathbf B_p^T = k_BT\left(\mathbf A_p+\mathbf A_p^T\right). \]

The extra freedom in \(\mathbf A_p\) makes the thermostat frequency-selective. PIGLET uses this freedom in the path-integral normal modes: instead of merely accelerating equilibration, it fits colored-noise matrices so that finite-\(P\) bead fluctuations reproduce selected quantum fluctuations much better than canonical finite-\(P\) PIMD. This is useful for equilibrium structural and kinetic-energy observables, but it should not be read as an exact real-time dynamics method.

In the harmonic test below, this idea is reduced to its simplest transparent form. For each bead number, the canonical finite-\(P\) value has a known bias. A toy PIGLET-style target asks what configurational covariance scale \(s_P=\langle V\rangle_{\mathrm{exact}}/\langle V_P\rangle\) would be needed to remove that one-observable bias. Real PIGLET does this with fitted colored-noise matrices rather than a scalar rescaling, but the diagnostic makes the role of covariance shaping explicit.

Workflow

  1. Use the same harmonic oscillator as Part I: \(m=\hbar=1\), \(\beta=2\), \(\omega_0=4\).
  2. For \(P=32\), compute the normal-mode frequencies \(\Omega_k\) and compare the thermostat matching ratio \(\gamma_k/(2\Omega_k)\) for one fixed white friction and for \(\gamma_k=2\Omega_k\).
  3. For \(P=1,\ldots,64\), compute the canonical finite-\(P\) bias of the harmonic potential estimator.
  4. Compute the one-observable covariance scale \(s_P=\langle V\rangle_{\mathrm{exact}}/\langle V_P\rangle\) that would remove that bias in a toy PIGLET-style target.
  5. Interpret the two diagnostics separately: PILE explains sampling efficiency; PIGLET explains deliberate finite-\(P\) covariance shaping.

Interpretation

The figure should be read as two different answers to two different problems. PILE says: if the ring polymer has many artificial normal-mode frequencies, thermostat them with mode-local frictions so the trajectory decorrelates efficiently. PIGLET says: if canonical finite-\(P\) PIMD still underestimates a selected quantum fluctuation, use a fitted colored-noise covariance to compensate for that finite-bead error. The price is specificity: a fitted GLE matrix is designed for a target frequency window and target observables, so one should check the observable being reported.
